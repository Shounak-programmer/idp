# Smart Traffic System

A multi-application system designed to reduce ambulance delays using AI-assisted traffic management and hornless communication.

## Modules

- **Driver App**: Radar-based hornless communication and spatial audio alerts.
- **Ambulance App**: Navigation with signal pre-emption and clearance broadcasting.
- **Admin Dashboard**: Traffic control center for monitoring and manual overrides.
- **Backend**: Firebase Functions for routing, notifications, and case management.
- **Shared**: Common utilities, simulators, and configuration.

## Setup

1. Install dependencies in root and all sub-folders.
2. Configure environment variables (see `.env.example`).
3. Run the simulator to generate traffic data.
4. Start the applications.

See `Docs/demo-instructions.md` for detailed deployment and usage instructions.
